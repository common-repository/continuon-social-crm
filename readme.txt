This plugin helps you add a Continuon social CRM banner to your Wordpress site.

== Description ==

[Homepage](https://app.continuon.co/) | [Documentation](https://app.continuon.co/knowledgebase/social-crm) | [Support](https://app.continuon.co/knowledgebase/social-crm)

This plugin will display a Continuon social CRM banner in your WordPress site by using Wordpress widgets.

More info and screenshots [Click Here](https://app.continuon.co/knowledgebase/social-crm/about-social-crm)

### Continuon Social CRM Plugin for WordPress

*   You must have a Continuon account to implement social CRM banners.
*   Once you've created a Continuon account and a social CRM banner, you can implement this plugin and enter your Continuon Brand ID and Banner ID to load the banner wherever you like.


### Features of Continuon Social CRM Plugin

*   User friendly and simple to implement.
*   Can be used to connect to your Continuon account and load a social CRM banner on any page in your site.
*   Include a Continuon social CRM banner in your posts & pages by using ```[continuon-social-crm]``` shortcode.

## Installation 

### Follow these instructions to install the plugin

### Automatic installation

1. Search for Continuon Social CRM Login through 'Plugins > Add New' interface.
2. Find the plugin by Typing Continuon and click on the 'Install Now' button.
3. Then activate the Continuon Social CRM Plugin.
5. Go to your Appearance => Widgets page.
6. Drag the Continuon Social Banner widget in to the sidebar or footer. Enter your brand ID (XXXX.continuon.co) and banner ID.
7. If you would like to install it in the content of a page, we recommend making use of a plugin that can generate a shortcode for you (such as https://wordpress.org/plugins/widget-shortcode/).


### Manual installation

1. Login in your WordPress admin panel.   
2. Navigate to Plugins => Add New from left menu bar.
3. Download the Plugin [here](https://app.continuon.co/static/plugins/social-plugin-wordpress.zip)
4. Click on the "Upload Plugin" at the top.
5. Choose the zip folder that you just downloaded, and click "Install Now".
6. Once uploaded, activate the plugin.
7. Go to your Appearance => Widgets page.
8. Drag the Continuon Social Banner widget in to the sidebar or footer. Enter your brand ID (XXXX.continuon.co) and banner ID.
9. If you would like to install it in the content of a page, we recommend making use of a plugin that can generate a shortcode for you (such as https://wordpress.org/plugins/widget-shortcode/).
